# MailForge Roadmap

## v1.0 — Current MVP

- [x] Contact management with CSV/XLSX/JSON/TXT import
- [x] Contact lists and tags
- [x] Global suppression list
- [x] SMTP sender accounts with multiple rotation modes
- [x] Email warmup engine with auto-advance and auto-pause
- [x] Campaign builder with HTML editor
- [x] BullMQ-based sending engine with rate limiting
- [x] Open/click/unsubscribe tracking
- [x] SPF/DKIM/DMARC/MX deliverability checks
- [x] Rules-based recommendation engine
- [x] Analytics dashboard with charts
- [x] In-app notifications
- [x] Audit activity log
- [x] Role-based access (Admin / Manager / Viewer)
- [x] JWT auth with refresh token rotation
- [x] Docker Compose deployment

## v1.1 — Planned

- [ ] A/B subject line testing for campaigns
- [ ] Re-engagement segment automation
- [ ] Contact custom fields UI
- [ ] Template variable preview with test data
- [ ] Bulk CSV export of contacts
- [ ] Campaign duplicate/clone
- [ ] Two-factor authentication (TOTP)
- [ ] Email preview across clients (via Litmus API)
- [ ] Webhook outbound notifications

## v1.2 — Planned

- [ ] Drip/sequence campaigns (multi-step automations)
- [ ] Dynamic segments (saved filter sets)
- [ ] Contact scoring rules (automated based on behavior)
- [ ] S3/MinIO integration for file storage
- [ ] IMAP inbox monitoring for replies and bounces
- [ ] DKIM signing per sender domain
- [ ] Multi-tenant support (organizations)
- [ ] API key authentication for external integrations
- [ ] Zapier / Make integration

## v2.0 — Future

- [ ] Visual drag-and-drop email builder
- [ ] AI-powered subject line suggestions
- [ ] Predictive send-time optimization
- [ ] Advanced deliverability scoring (real-time)
- [ ] Dedicated IP warmup workflows
- [ ] Transactional email API (single sends via REST)
- [ ] Mobile app (React Native)
